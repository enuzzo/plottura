Phase 8 · Nerd Font Icons

Nerd Font Icons

250+ curated glyphs · 10,000+ available. Two-tier system: VibemilkIcons (subset, ~40KB) for production, full Symbols Nerd Font for exploration. Token-driven, theme-aware, golden-ratio scaled.

Quick start

Three ways to use icons

1. Unicode-first

Paste the hex entity directly. Maximum control. 

<span class="vm-nf" aria-hidden="true">
  &#xeb49;
</span>

2. Named class

Semantic, readable, no codepoint memorization. 

<span class="vm-nf vm-nf-lightning"
  aria-hidden="true"></span>
3. With modifiers

Combine size + color modifiers for precise control. 

<span class="vm-nf vm-nf-lightning
  vm-nf--2xl
  vm-nf--accent"
  aria-hidden="true"></span>
Size scale

Golden ratio progression

 --sm · 0.75em
 --md · 1em
 --lg · 1.272em (sqrt phi)
 --xl · 1.618em (phi)
 --2xl · 2.618em (phi^2)
All sizes derive from the golden ratio. Base = 1em inherits from parent font-size, so icons scale naturally with text.
Color variants

Token-driven color modifiers

Integration

Loading the icon font

Production (subset)

VibemilkIcons.woff2 (~40KB) contains only the 250+ curated glyphs. Loaded automatically via vibemilk-core.css. 

/* Already in tokens.css */
--font-icons: "VibemilkIcons", "Symbols Nerd Font";
Exploration (full)

Install Symbols Nerd Font Mono system-wide for 10,000+ glyphs. The fallback in --font-icons picks it up automatically. 

/* Use any Nerd Font codepoint */
<span class="vm-nf">&#xf10fe;</span>
Token reference: var(--font-icons) · Base class: .vm-nf · Full icon CSS: css/core/icons.css · See Integration Guide for setup details.
