Themes

Theme contract: fast, modular, maintainable

Theme File Use
Vibemilk Default core/tokens.css production baseline
Cyber Grid themes/cyber-grid.css high-contrast terminal UI
Acid Pop themes/acid-pop.css neon/candy accents
Sakura Spring themes/sakura-spring.css metro-signage palette
Mono Brutalist themes/mono-brutalist.css hard mono editorial
Mint Protocol themes/mint-protocol.css milk-and-mint UI with lilac/blush contrast
Mint Protocol ships with the Herbal Harmony palette (#ADEBB3, #9EB8A0, #58855C, #0D3311) as its base, then layers split/triadic lilac and blush contrasts on top. Typography stays on Encode Sans Semi Expanded in weights 400 and 500.
How to add a new theme
/* css/themes/arctic-noise.css */
:root[data-theme="arctic-noise"] {
  --theme-id: 'arctic-noise';
  --accent-primary: #8BE8FF;
  --bg-surface: #0F1E2A;
  --font-family: 'Chakra Petch', sans-serif;
}
Hook it into the JS manager
const THEME_MANIFEST = {
  "vibemilk-default": null,
  "arctic-noise": "css/themes/arctic-noise.css"
};

Vibemilk Default

Production baseline. Balanced dark palette with purple-blue accents and clean typography.

Cyber Grid

High-contrast terminal aesthetic. Neon green on deep black with scan-line textures.

Acid Pop

Neon candy accents on dark surfaces. Bold, energetic palette for creative interfaces.

Sakura Spring

Metro signage palette. Japanese transit system colors with clean geometric accents.

Mono Brutalist

Hard mono editorial. Black-and-white with sharp typography and minimal decoration.

Mint Protocol

Milk-and-mint UI with lilac/blush contrast. Herbal Harmony base palette.
