How should the user switch between 2D and 3D?

The 3D mode adds pitch + buildings + terrain to the map. Where does the toggle live?

A

New "3D View" section in sidebar

A dedicated sidebar accordion section with: 3D on/off toggle, pitch slider (0°–85°), building height multiplier, terrain on/off, lighting angle. Full control panel — consistent with how Layers and Map Settings already work.

B

Button in the map controls bar

A "3D" toggle button next to Recenter/Edit Map in the bottom notch area. Quick access, one click to flip. Pitch slider appears in the floating controls when 3D is active (like zoom slider appears during Edit).

C

Both — button for quick toggle, sidebar for fine control

A "3D" button on the canvas for fast switching. Detailed controls (pitch, terrain, lighting) in a sidebar section. Best of both: quick access + deep customization. Similar to how "Edit Map" enables floating controls.
