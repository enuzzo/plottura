How this works technically

MapLibre GL JS already supports this natively!

MapLibre Pitch

MapLibre has a built-in pitch property (0°–85°). Setting pitch: 60 tilts the camera to show the map in perspective — exactly like Google Maps 3D tilt.

3D Buildings

OpenMapTiles vector tiles include building height data. MapLibre can render fill-extrusion layers — buildings automatically become 3D boxes. Zero extra data needed.

Terrain / Hills

MapLibre v4+ supports terrain with DEM tiles (e.g., MapTiler terrain). The map surface deforms to show real elevation — mountains, valleys, coastlines gain depth.

No Extra Library

No Three.js, no CSS 3D transforms for the map itself. The poster frame stays flat. MapLibre's own WebGL renderer handles all 3D. We just need to toggle pitch and add extrusion layers.
