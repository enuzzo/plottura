# Plottura — From plot to print

A cartographic poster design tool. Create print-ready map posters for any place on Earth, entirely in your browser.

## Features

- Location search with autocomplete (powered by Nominatim)
- Dozens of curated map themes with custom colour palettes
- Layer control for fine-grained map styling
- Draggable markers with built-in and custom icons
- Full typography controls (Google Fonts, size, colour, layout)
- Export as high-resolution PNG, SVG, or PDF

## Getting Started

```bash
bun install
bun run dev
```
